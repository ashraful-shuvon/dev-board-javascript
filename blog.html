Question-1: What are the different ways to select an element in the DOM?

There are different ways of selecting an element in the DOM. They are used depending on the different scenarios.

  1. getElementById()
  2. getElementsByClassName()
  3. getElementsByTagName()
  4. querySelector()
  5. querySelectorAll()
Question-2: What is the difference between innerHTML, innerText, and textContent ?
These are the properties of JavaScript. They are used to manipulate the HTML elements. But they are different from each other. innerHTML retrives, also can set the HTML elements, including tags. innerText only retrives and can set the main text of a HTML tag, ignoring HTML tags and hidden content. On the other hand, textContent can retrive all the text content including the hidden contents from inside of a HTML tag, but it ignores the HTML tag.
Question-3: What is event delegation in the DOM?
Event delegation is a JavaScript technique which is used where a single event listener is attached to a parent node to handle the events from its child nodes. Instead of using event listener to every child node it utilizes the event bubbling technique to listen the events from its child nodes to parent node.
Question-4: What is event bubbling in the DOM?
Event Bubbling is an event in the DOM, it starts from the target element to its parents, then all its ancestors which are on the way from bottom to top, to the root node or parent.
Question-5: How do you create, add, and remove elements using JavaScript?
Create element: document.createElement(tagName). Add element: parentNode.appendChild(childNode). Remove element: parentNode.removeChild(childNode)
